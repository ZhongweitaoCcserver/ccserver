What is Ccserver?
-------------------------------------------------------------------
Ccserver is a concurrency server demo, multithreaded, runs on Linux. 
Compiler environment: C++ 11, google protobuf, google tcmalloc.
It can be compiled success on centos7 now.(Linux version 3.10.0-327.el7.x86_64)
